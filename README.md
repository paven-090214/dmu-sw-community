# dmu-sw-community

## 프로젝트 참여자 및 역할

- 추교현 : 프론트 UI, 페이지 구조, CSS

- 안수환 : React 보조, Spring Boot, 홈서버 배포

- 이시현 : DB 설계, ERD, SQL

## 구현 기능

### 인증 기능

- 로그인, 회원가입, 로그아웃

### 비로그인 유저

- 동양미래대학교 공지
- 컴소과 공지
- 잡담 게시판
- QNA 게시판

### 로그인 유저

- 잡담, QNA 게시판 글 쓰기
- 잡담, QNA게시판 댓글, 대댓글..
- 잡담, QNA게시판 개추, 비추, 스크랩
- 학교, 학과 공지 스트랩, 개추, 비추
